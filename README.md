# Secure Vote Ledger

A Blockchain-Based E-Voting System

## Team Members

## Tech Stack
- React (Frontend)
- Spring Boot (Backend)
- MySQL (Database)
- Blockchain (Planned)

## Features Implemented
- User Login & Registration
- Protected Routes
- Constituency-wise Voting
- Results Visualization
- MySQL Integration

## Project Structure
- frontend/ -> React application
- backend/ -> Spring Boot application
